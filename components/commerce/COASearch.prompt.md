Searchable Certificate-of-Analysis lookup by batch number. Anchors the Test Results / COA page.

```jsx
<COASearch data={batches} />
```

- Live-filters as you type (mono input). Results table shows batch, product, purity, Verified badge, and a PDF/Drive link.
- Empty state guides the user to their vial label. Compliance-safe — purity + verification only, no claims.
