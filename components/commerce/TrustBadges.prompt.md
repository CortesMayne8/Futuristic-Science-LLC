The five brand trust pillars as an icon row. Place under the hero and on product pages.

```jsx
<TrustBadges />                          // all five, light
<TrustBadges variant="ink" size="sm" />  // on a dark band
```

- Default items: 99%+ Purity, Third-Party Tested, Batch COAs, Measured Accuracy, Fast Shipping.
- Built-in Lucide-style icons (shield, flask, file-badge, ruler, truck). `variant="ink"` for dark backgrounds.
